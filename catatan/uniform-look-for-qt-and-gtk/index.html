Uniform Look for Qt and GTK Applications


Tips and Trick

Tips and Trick kali ini kita akan membuat tampilan dari apps yang berbasis qt menggunakan themes dari gtk, dalam kasus ini kita menggunakan themes gtk2.

Gtk

Gtk adalah kependekan dari GIMP ToolKit, GIMP ?, GIMP (GNU Image Manipulation Program), disematkan nama GIMP dikarenakan project ini berjalan bersamaan dengan project GIMP. Gtk digunakan untuk membuat tampilan atau user interfaces berbasis grafis atau sering disebut GUI. Lebih jelasnya langsung cek disini

Qt

Sama halnya dengan Gtk, Qt bisa kita baca “cute” juga termasuk ToolKit yang digunakan untuk membangun user interfaces berbasis grafis, untuk lebih jelasnya langsung saja ke website resminya disini

Persiapan

Untuk membuat qt menggunakan themes dari gtk2 kita memerlukan package yang bernama qt5ct dan qt5-styleplugins, langsung saja install package yang digunakan.

$ sudo pacman -S qtct qt5-styleplugins

Setelah package terinstall tambahkan environment QT_QPA_PLATFORMTHEME=qt5ct, pada file /etc/environment

Berikut contoh tampilan apps qt yang menggunakan themes gtk2

  • Pastikan themes yang di pakai support dengan gtk2.

Kesimpulan

Tips and Tricks ini cocok di terapkan bagi mereka yang ingin menyeragamkan semua tampilan aplikasinya tanpa memeikirkan themes untuk qt.
