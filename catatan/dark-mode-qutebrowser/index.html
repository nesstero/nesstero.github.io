Dark Mode Qutebrowser


Dark mode qutebrowser

Untuk menggunakan dark mode pada qutebrowser ada beberapa cara diantaranya :

  • content.user_stylesheets
  • colors.webpage.darkmode.enabled
  • colors.webpage.preferred_color_scheme
  • greasemonkey Pada kasus ini kita akan mencoba menggunakan cara yang terakhir yaitu dengan greasemonkey atau monkey script dimana cara ini lebih mudah dari ketiga cara lain diatas dan hasilnya juga lumayan bagus.

Membuat script js

Disini kita akan membuat user script js untuk membuat dark mode dengan memanfaatkan dark reader , pertama buat terlebih dahulu file user script jsnya, dengan nama dark mode.user.js dalam folder ~/.config/qutebrowser/greasemonkey/, kedua pastekan script berikut kedalam file dark mode.user.js yang telah kita buat :

// ==UserScript==
// @name          Dark Mode
// @namespace     DarkReader
// @description	  Inverts the brightness of pages to reduce eye strain
// @version       4.7.15
// @author        https://github.com/darkreader/darkreader#contributors
// @homepageURL   https://darkreader.org/ | https://github.com/darkreader/darkreader
// @run-at        document-end
// @grant         none
// @include       http*
// @require       https://cdn.jsdelivr.net/npm/darkreader/darkreader.min.js
// @noframes
// ==/UserScript==

DarkReader.enable({
	brightness: 100,
	contrast: 90,
	sepia: 10
});

Untuk value brightness, contrast, dan sepia bisa di sesuaikan dengan kenyamanan masing - masing. Terakhir reload script monkeynya dengan cara :greasemonkey-reload
