Tips and Tricks

Tips and Trick kali ini kita akan membahas tentang Disqus yang belakangan ini membuat feature iklan pada disqus free dan tidak bisa di-disable, mau tidak mau untuk mendisable iklan harus upgrade dan tentunya berbayar ðŸ˜­.

Solusi

  • Solusi pertama tentu saja upgrade ðŸ˜Š ke versi berbayar
  • Solusi kedua menggunakan script jQuery
  • Solusi ketiga menggunakan css

Tentunya kita tidak akan membahas solusi pertama, yang akan kita bahas hanya solusi kedua dan ketiga

Menggunakan script jQuery

Untuk hide ads dari disqus kita perlu menambahkan script jQuery ke dalam website kita, seperti berikut:

const disqus = jQuery('#disqus_thread');

disqus.ready(function() {
  setTimeout(function() {
    if (disqus.children().length >= 3) {
      const comments = disqus.find('iframe:nth-child(2)').detach();
      disqus.empty().append(comments);
    }
  }, 2000);
});

Menggunakan css

Cukup menambahkan satu baris css untuk hide ads dari disqus

iframe[src*="ads-iframe"] { display: none; }
