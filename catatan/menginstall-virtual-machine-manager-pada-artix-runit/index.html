Install Qemu Virtual Machine Manager On Artix Runit


Qemu

Qemu merupakan machine emulator dan virtualizer.

Kvm

KVM kependekan dari Kernel-based Virtual Machine adalah modul virtualisasi di kernel Linux yang memungkinkan kernel berfungsi sebagai hypervisor.

Menginstall qemu dan virtual manager

$ sudo pacman -S qemu virt-manager libvirt-runit vde2 dnsmasq ebtables bridge-utils openbsd-netcat
  • Jika package tidak di temukan aktifkan repo dari arch linux extra, dan community dan pastikan repo artix lib32 aktif

Konfigurasi libvirtd

konfigurasi pada file /etc/libvirt/libvirtd.conf uncomment pada baris berikut :

 unix_sock_group = "libvirt"
 unix_sock_rw_perms = "0770"

Menambahkan group libvirt

$ sudo usermod -a -G libvirt $(whoami)
$ newgrp libvirt

Mengaktifkan service libvirtd, virtlockd, dan virtlogd

$ sudo ln -s /etc/runit/sv/libvirtd /etc/runit/runsvdir/default
$ sudo ln -s /etc/runit/sv/virtlockd /etc/runit/runsvdir/default
$ sudo ln -s /etc/runit/sv/virtlogd /etc/runit/runsvdir/default
$ sudo reboot
