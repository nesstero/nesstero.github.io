Arch linux

Arch Linux ?

Apa itu arch linux ðŸ¤”ðŸ¤”, kita akan bahas sedikit apa itu arch linux. Arch linux adalah salah satu dari sekian banyak distribusi gnu/linux yang menganut prinsip KISS katanya ðŸ˜‚. Arch linux menggunakan package manager yang di sebut pacman. Arch linux juga memiliki wiki/dokumentasi yang lumayan lengkap jika di bandingkan dengan distribusi yang lainnya sampai - sampai memiliki jargon RTFM. Untuk lebih jelasnya mengenai arch linux silahkan kunjungi website resminya disini .

Persiapan

Langsung saja ke topik utama kita Bagaimana Cara Menginstall Arch Linux ?. Pertama pastikan sudah memiliki bootable usb arch linux, untuk file isonya bisa di unduh langsung pada website resmi arch linux disini . Dalam pembahasan ini kita tidak akan membahas bagaimana caranya mempartisi HDD/SSD, disini kita memiliki 3 partisi, partisi swap, root, dan partisi home kurang lebih seperti ini :

  • /dev/sda1 untuk partisi swap ukurannya bisa disesuaikan dengan kapasitas RAM.
  • /dev/sda2 untuk partisi root rekomendasi kasih partisi root minimal 50 GB.
  • /dev/sda3 untuk partisi home bebas.

Memformat dan mengaitkan partisi


  mkswap /dev/sda1
  mkfs.ext4 /dev/sda2
  swapon /dev/sda1
  mkdir -p /mnt/home
  mount /dev/sda2 /mnt
  mount /dev/sda3 /mnt/home
  • mkswap digunakan untuk memformat partisi swap
  • mkfs.ext4 digunakan untuk memformat partisi dengan format Ext4
  • swapon digunakan untuk mengaktifkan swap
  • mkdir -p /mnt/home untuk membuat parent home direktori
  • mount digunakan untuk mengaitkan partisi

tambahan bagi pengguna uefi :


  mkfs.fat -F 32 <partisi-efi>
  mkdir -p /mnt/{boot/efi,home}
  mount <partisi-efi> /mnt/boot/efi
  • mkfs.fat -F 32 untuk memformat partisi efi

Proses Instalasi


  iwctl
    iwd# device list
    iwd# station wlan0 scan
    iwd# station wlan0 get-networks
    iwd# station wlan0 connect `<nama-SSID>`
    iwd# exit
  pacstrap /mnt base base-devel linux linux-firmware nano
  genfstab -U /mnt >> /mnt/etc/fstab
  • iwctl/iwd adalah wireless daemon
  • pacstrap /mnt …… digunakan untuk menginstall base paket ke direktori /mnt
  • genfstab digunakan untuk mengenerate file fstab

Konfigurasi

Hal - hal yang perlu di konfigurasi setelah menginstall arch linux :


  arch-chroot /mnt
  timedatectl set-timezone Asia/Jakarta
  • arch-chroot di gunakan untuk mengubah direktori root
  • timezone (WIB)

  nano /etc/locale.gen
  locale-gen
  echo LANG=en_US.UTF-8 > /etc/locale.conf
  export LANG=en_US.UTF-8
  • locale
  • uncomment en_US.UTF-8 dan id_ID.UTF-8 pada file locale.gen

  echo myarch > /etc/hostname
  touch /etc/hosts
  nano /etc/hosts
  • touch digunakan untuk membuat sebuah file
  • isi dari file host :

  127.0.0.1	localhost
  ::1 localhost
  127.0.1.1	myarch
  • myarch bisa dirubah dengan nama host lainnya

  passwd
  pacman -S grub
  grub-install /dev/sda
  grub-mkconfig -o /boot/grub/grub.cfg
  • passwd digunakan untuk mengatur password root
  • pacman -S digunakan untuk menginstall paket
  • grub adalah boot loader
  • tambahan bagi pengguna uefi:

  pacman -S grub efibootmgr
  grub-install --target=x86_64-efi --bootloader-id=GRUB --efi-directory=/boot/efi
  grub-mkconfig -o /boot/grub/grub.cfg

Menambahkan Username


  useradd -m -g users -G wheel -c "Ara" ara
  passwd ara (membuat password untuk user ara)
  pacman -S sudo
  nano /etc/sudoers
  • useradd digunakan untuk menambahkan user baru
  • sudo adalah alternatif dari su untuk menjalankan perintah sebagai root
  • masukan user baru ke dalam file sudoers :

  root ALL=(ALL) ALL
  ara All=(ALL) ALL

Menginstal Desktop Environment


 pacman -S xorg plasma
 systemctl enable sddm.service
 systemctl enable NetworkManager.service
  • plasma merupakan Desktop Environment buatan KDE

Reboot


  exit
  reboot
